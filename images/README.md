## ملاحظات

1. **الصيغ المدعومة:** PNG, JPG, SVG, WebP
2. **الحجم:** يفضل ضغط الصور للحصول على أفضل أداء
3. **الجودة:** استخدم صور عالية الجودة للحصول على مظهر احترافي
4. **الخلفية:** يفضل استخدام خلفية شفافة للشعار والأيقونات
